# Bulls_and_Cows

Projekt JetBrains Academy

Jest to gra w której użytkownik szuka liczby losowo generowanej przez program.
Liczba może być o długości i symbolach 1-36, każdy symbol jest unikatowy, więc symbol>=długość 

Przykłady działania programu:

![image](https://user-images.githubusercontent.com/92478936/203866534-94418571-fa79-48ba-bde0-36baeab0c5f4.png)

Oczywiście kod jest ukryty, ale dla lepszego wyjaśnienia go pokazałem. W tym przypadku program generuje losowy kod składający się ze znaków (0-9, a-y)
o długości 25. Po każdej rundzie jest pokazana ocena w której jest umieszczona ilość bull - Liczba na tym samym miejscu co w kodzie, oraz cow - Liczba która jest taka sama jak w kodzie lecz na innym miejscu.
